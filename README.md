# Alex Dev Portfolio
> Professional personal portfolio built for CodeAlpha Frontend Internship

## Features
- **Single Page Design** — Smooth scrolling between sections
- **Responsive Layout** — Mobile-first approach with hamburger menu
- **Typing Animation** — Dynamic hero text effect
- **Scroll Animations** — Intersection Observer for fade-in effects
- **Animated Skill Bars** — Progress bars animate when scrolled into view
- **Project Cards** — Hover overlay with live links and code links
- **Timeline Resume** — Experience and education in vertical timeline
- **Contact Form** — Validation with success feedback
- **Scroll to Top** — Floating button appears after scrolling
- **Active Navigation** — Auto-updates based on scroll position

## Sections
1. **Hero** — Introduction with CTA buttons and social links
2. **About** — Bio with animated stats counters
3. **Skills** — Technical skills with animated progress bars
4. **Projects** — Featured work with tags and hover effects
5. **Resume** — Experience and education timeline
6. **Contact** — Contact info and functional form

## Tech Stack
- HTML5 Semantic markup
- CSS3 (Grid, Flexbox, Custom Properties, Animations)
- Vanilla JavaScript (ES6+)
- Font Awesome icons
- Google Fonts (Inter + Playfair Display)

## How to Run
Open `index.html` in any modern browser. No build step required.
